Idea and thought for Cubic Newton

Cubic Newton

After having a conversation with Bo.J, I’ve change my direction to Cubic Newton which is a second-order algorithm. I’ve read Cubic Regularization of Newton written by Nestrov, and there are kinds of preponderence of CR:

  • CR has a faster global convergence rate than Gradient Descent and local quadratic convergence.
  • CR can jump out of Saddle point.
  • CR have a great result in some weak convex problem such as Star Convex
  • CR can keep necessary second order optimality condition.

Difficulty of this algorithm is its subproblem, as this subproblem is a 1 dimenstional eqution with some tricky nonlinear function. The Tradtional method is Trust region method (but acutally I don’t really know this method)

Adaptive Cubic Newton

Then A new method is Adaptive Cubic Regularization (ACR), this method is much relative to Nesterov’s method and there are some new tricky.

This paper suggests that we should adaptively control the estimation of Lipschitz constant corresponding to the result of current update step.

Besides, this paper use Krylov Subspace and a use estimatier of Hessian matrix to implement this job.

How to learn it?

First problem is how to guarantee the result satisfy second order principle. Some other difficulies concludes how to train? how to design objective function? etc.
